# Kidney Stone Prediction System

# Project Overview
This is a machine learning-based desktop application that predicts kidney stone risk using patient medical data.

# Technologies Used
- Python
- Tkinter
- MySQL
- Scikit-learn
- ReportLab (python-library)

# Features
- Patient mediacl data entry
- Kidney stone prediction
- MySQL database storage
- PDF report generation
- Prediction statistics

# Model Accuracy
- 86%

# How to Run
1. Install required libraries.
2. Import the MySQL database.
3. Run Kidney_Stone_Prediction.py
